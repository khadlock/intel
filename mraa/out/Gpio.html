Gpio

new Gpio(pin, owneropt, rawopt) → {Object}

Gpio Constructor - API to General Purpose IO
Parameters:
Name Type Attributes Default Description
pin number The pin number to use
owner boolean <optional>
 true Set pin owner, default behaviour is to 'own' the pin if we exported it. This means we will close it on destruct. Otherwise it will get left open. This is only valid in sysfs use cases
raw boolean <optional>
 false Raw pins will use gpiolibs pin numbering from the kernel module. Note that you will not get any muxers set up for you so this may not always work as expected.
